Flex box funciona em dois eixos. O eixo padrao da esquerda para direita. E o eixo cruzado de cima para baixo.
Chama-se o flex na div principal.

flex-wrap

Para quebra de linha depois que o eixo principal for totalmente ocupado. utilizamos o FLEX WRAP que vem nativamente desativado( no wrap). para ativar: flex-wrap: wrap;

align and justify

Para alinhamento utilizamos as propriedades flex-direction, que altera o modo de alinhamento. o row-reverse por exemplo altera a o alinhamento do eixo principal da direita para esquerda.
Para inverter o eixo principal com o eixo cruzado utilizamos a propriedade column (flex-direction: column;)
A propriedade align itens alinha os elementos no eixo cruzado, de cima para baixo. a configuração padrao dessa propriedade é a baseline. align-items: baseline; podemos alinhar pelo centro utilizando a proriendade center. A propriedade flex-end alinha no final da div.
Para alinharmos no eixo princioal, da esquerda para direita. utilizamos justify-content. Para adionar um espaço exatamente igual entre cada elemento da div utilizamos: justify-contente: space-between; nesse caso não há distanciamento entre o elemento e a margin.
O space-around coloca um espaço entre os elementos e o adiocna um espaço entre as margins.

flexGrow flexShrink flexBasis

>flex-basis define a largura da div
>flex-grow é o fator de crescimento da minha div
