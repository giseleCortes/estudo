Flex box funciona em doi eixos. O eixo padrao da esquerda para direita.

E o eixo cruzado de cima para baixo

chama-se o flex na div principal.
Para quebra de linha depois que o eixo principal for totalmente ocupado. utilizamos o FLEX WRAP que vem nativamente desativado( no wrap). para ativar: flex-wrap: wrap;
Para alinhamento utilizamos as propriedades flex-direction, que altera o modo de alinhamento. o row-reverse por exemplo altera a o alinhamento do eixo principal da direita para esquerda.
Para inverter o eixo principal com o eixo cruzado utilizamos a propriedade column (flex-direction: column;)
