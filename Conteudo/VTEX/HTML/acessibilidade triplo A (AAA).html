A - Fornece uma breve descrição

AA - A + legenda para audio e vídeo

AAA - A + AA + linguagem de sinais

ARIA é uma especificação da W3C para aprimorar a acessibilidade (A11Y) que é definida através dos atributos de um elemento HTML

exemplo

Acessibilidade LANDMARKS

  • role - definem o que um elemento é ou faz.
  • state - propriedade especiais que definem condições atuais dos elementos
  • properties - descreve a relação com outros elementos e emes não mudam uma vez que são definidos
